The project uses Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn to preprocess the dataset and generate meaningful insights about AI adoption trends.
🚀 Features
Data loading and preprocessing
Handling missing values and duplicate records
Exploratory Data Analysis (EDA)
Univariate and bivariate analysis
Correlation analysis between variables
Data visualization using charts and heatmaps
Insight generation about AI adoption trends

Python

Jupyter Notebook

View first few rows of dataset

Identify industries with high AI adoption
